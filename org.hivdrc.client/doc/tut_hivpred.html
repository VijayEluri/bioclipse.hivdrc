Using the HIV Protease Susceptibility Wizard


Open the wizard

In the main menu, chose the alternative "Web services -> HIV Protease Susceptibility Prediction".

The Wizard

The HIV Protease Susceptibility Wizard opens. Paste an HIV protease sequence and click Finish.




The Result View

The Result View opens and displays the susceptibility for 7 drugs. The Bioclipse console also echoes the results for simpe copy and paste.
