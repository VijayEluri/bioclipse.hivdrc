User guide

Configure XMPP account

You need an XMPP account (also known as Jabber ID, or simply JID) in order to use the HIVDRC services. You can register for an account at any public XMPP service. When you have obtained a JID, enter this information in the Bioclipse Preferences from menu Window > Preferences and input username and password on the XMPP tab.


Access service from the GUI

In the Bioclipse Explorer you can right-click on sequences (with extension .fasta and .seq) and invoke the HIVDRC services. Note that PIs should be 99 aa long, and NRTIs should be 240 aa long.


Access service from scripts

It is straightforward to invoke the HIVDRC services from scripts in Bioclipse. To invoke a single service, open up the JavaScript console from the menu Window > Show View > Other... and select Scripting > JavaScript console. In the console, type:

   help hivdrc

This shows the available commands for consuming the services.


To predict a PI, enter:

   hivdrc.predict("PQITLWQRPLVTIKIGCQLKEALLDTGADDTVLEEANLPGRWKPKMIGGIGGFIK
VRQYDQILIEICGHKAIGTVLVGPTPVNIIGRNLLTQIGCTLNF")


To predict a NRTI, enter:

   hivdrc.predict("PISPIETVPVKLKPGMDGPKVKQWPLTEEKIKALVEICTEMEKEGKISKIGPENPYN
TPVFAIKKKDSTKWRKLVDLRELNKRTQDFWEVQLGIPHPAGLKKKKSVTVLDVGDAYFSVPLDKDFRKYTAFTIPSIN
NETPGIRYQYNVLPQGWKGSPAIFQSSMTKILEPFRKQNPDIVIYQYMDDLYVGSDLEIGQHRTKIEELRQHLLRWGFT
TPDKKHQKEPPFLWMGYELHPDKWT")


A sample JavaScript for invoking HIVDRC functionality in Bioclipse is available as a Gist (versioned snippet), with number 403896. Bioclipse can easily download these scripts. In the JavaScript console, type:

   gist.download(403896)

The file "Gists403896.0.js" is downloaded in a folder "Gists" in the Bioclipse Explorer. Open this file, and click the "Run" button in the toolbar to execute the script.
